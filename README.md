# 素因数分解ツール
![test](https://github.com/KazukiFujita666/robosys2025/actions/workflows/test.yml/badge.svg)

標準入力から数字を読み込み、素因数分解したものを出力します.

##インストール方法
'''
$ git clone git@github.com:KazukiFujita666/robosys2025.git
$ cd robosys2025
$chmod +x factorize
'''
